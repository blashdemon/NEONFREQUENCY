# 🌌 NEON FREQUENCY Linux Distribution

**NEON FREQUENCY** is an advanced, lightweight Linux distribution built entirely on **Debian Stable** and optimized to run with high efficiency on Android smartphones via **Termux** using **PRoot-Distro**.

Developed by **BLACK LEGACY**, this distribution provides portable, integrated environments tailored for developers, cybersecurity researchers, and digital forensics analysts without requiring root access (No Root Required).

## 🔮 Distribution Editions

**NEON FREQUENCY** comes in **5 specialized editions** to meet your exact needs based on your device hardware:

### 1. 🏠 NEON FREQUENCY Home
*   **Description:** The vanilla base and parent image of the distribution.
*   **Features:** Completely free of penetration testing or digital forensics tools. Extremely stable, lightweight, and perfect as a daily-drive software development and general-use environment. It serves as the solid foundation upon which all other editions are built.

### 2. ⚡ NEON FREQUENCY Security Lite (Sec-Lite)
*   **Description:** A fast and lightweight penetration testing edition.
*   **Features:** Contains essential network scanning tools, vulnerability assessment scripts, and core utilities. **Does NOT include the Metasploit Framework**, making it ideal for budget or mid-range devices and rapid on-the-go operations.

### 3. 🛡️ NEON FREQUENCY Security Pro (Sec-Pro)
*   **Description:** The ultimate advanced penetration testing arsenal.
*   **Features:** Packed with massive software packages and sophisticated offensive tools. **Fully includes the Metasploit Framework**, social engineering kits, advanced vulnerability scanners, custom encryption, and payload generation scripts like BD1.

### 4. 🔍 NEON FREQUENCY DF Lite (Forensics Lite)
*   **Description:** A light digital forensics and incident response (DFIR) edition.
*   **Features:** Tailored for lightweight evidence gathering, basic network analysis, and quick memory checks. Optimized to consume minimal system resources while handling fundamental forensic tasks.

### 5. 🔬 NEON FREQUENCY DF Pro (Forensics Pro)
*   **Description:** A comprehensive, heavy-duty mobile digital forensics laboratory.
*   **Features:** A robust and resource-heavy edition containing vast packages for deep system analysis, deleted file recovery, reverse engineering, advanced network analysis, and volatile memory (RAM) forensics.

## 🛠️ Prerequisites

Before installing, ensure the following core applications are installed on your Android device:
1. **Termux**
2. **PRoot-Distro** package for managing environments within Termux.

---

## 🌌 General Features

*   **Debian Stability:** Relying on Debian Stable repositories ensures an error-free, rock-solid user experience.
*   **Isolated & Secure:** Runs entirely inside a sandboxed PRoot environment, meaning it will never alter or harm your phone's native system files.
*   **High Performance:** Intelligent resource consumption optimized for ARM processors and smartphone RAM management.
*   **Premium Neon Theme:** Custom CLI interfaces and scripts styled with a unique cyber-neon aesthetic representing our team identity.
